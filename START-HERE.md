## 📞 ¿NECESITAS AYUDA?

### Problemas Comunes

**"No puedo instalar las dependencias"**
→ Ver sección Troubleshooting en INSTALACION-RAPIDA.md

**"No conecta a la base de datos"**
→ Verificar PostgreSQL corriendo y credenciales en .env

**"El puerto está en uso"**
→ Cambiar PORT en .env

**"Error con bcrypt"**
→ Instalar build-essential: `sudo apt install build-essential`
